# 수정 가이드 (Customization Guide)

이 가이드는 청첩장 내용을 수정하는 방법을 설명합니다.

## 1. 메인 사진 및 텍스트 (Hero 섹션)
**파일 위치**: `components/Hero.tsx`

- **배경 이미지**:
  - `<Image />` 컴포넌트의 `src` 부분을 변경하세요.
  - 내 컴퓨터의 사진을 쓰려면 `public` 폴더에 사진을 넣고 `/사진이름.jpg` 처럼 적으면 됩니다.
  - *주의*: 외부 인터넷 주소(URL)의 사진을 쓴다면 `next.config.ts` 파일에 설정을 추가해야 합니다.
- **텍스트**:
  - "김철수 & 이영희" 부분을 본인 이름으로 바꾸세요.
  - 날짜와 장소 텍스트도 수정하세요.

## 2. 갤러리 사진
**파일 위치**: `components/Gallery.tsx`

- **사진 목록**:
  - `const images = [...]` 부분을 찾으세요.
  - 여기에 사용할 사진들의 주소(URL)를 넣으면 됩니다.
  - 세로로 긴 사진(비율 2:3 정도)이 예쁘게 나옵니다.

## 3. 오시는 길 (지도)
**파일 위치**: `components/Map.tsx`

- **텍스트**:
  - "그랜드 웨딩홀"과 주소, 전화번호를 수정하세요.
- **지도 넣기**:
  - 현재는 회색 박스로 되어 있습니다.
  - 카카오맵이나 네이버지도에서 "공유하기" -> "HTML 태그 복사" 기능을 이용해 `<iframe>` 코드를 복사하세요.
  - `<div className="relative w-full h-[400px] ...">` 부분을 지우고 복사한 코드를 붙여넣으세요.
- **버튼 링크**:
  - "카카오맵", "네이버지도", "구글맵" 버튼의 `href` 주소를 실제 예식장 위치 링크로 바꿔주세요.

## 4. 하이드레이션 경고 (Hydration Warning)
**파일 위치**: `app/layout.tsx`

- 브라우저 확장프로그램(번역기 등) 때문에 발생하는 에러를 무시하도록 설정해두었습니다.
- `suppressHydrationWarning` 옵션이 켜져 있으니 안심하셔도 됩니다.
